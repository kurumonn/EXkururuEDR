# exkururuEDR

[Japanese README](README.md)
[4-stack demo note](README.4stack.md)

EXkururuEDR is the endpoint detection and response component of the EXkururu stack.
The public repository keeps the parts that are useful to evaluate openly: agent layout, event
normalization, contract usage, and local tooling.

## Public scope

- Linux agent skeleton
- Endpoint event normalization
- Shared event contract usage
- XDR export surface
- Local CLI and tests

Detection tuning, production thresholds, and detailed optimization notes are intentionally excluded from the
public distribution.
