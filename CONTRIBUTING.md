# Contributing to exkururuEDR

Thank you for contributing.

## Scope

- Keep this project independent from Django application code.
- Focus on endpoint telemetry, detection logic, and lightweight runtime behavior.

## Development rules

- Prefer small, reviewable pull requests.
- Add tests for new logic when practical.
- Keep backward compatibility for public interfaces unless explicitly versioned.

## Security and privacy

- Do not commit secrets, tokens, or private customer data.
- Use synthetic/redacted samples for test fixtures.
